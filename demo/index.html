Annoto × Kaltura Playkit v7 — wrapper plugin demo

This page proves the wrapper works the way a uiConf-bundled plugin would: Annoto is enabled purely through the player plugins config — there is no host-page Annoto script tag. The wrapper (registered below) loads plugin.js from inside the player. This is the same path that runs inside the Kaltura IFrame embed once the plugin is bundled into a uiConf.

Anonymous-only: just the API key, no user login/SSO.

Defaults point at Annoto's Kaltura test partner. Replace with your own to test your account.
